
## wincoe

Originally located here: https://github.com/workturnedplay/wincoe  
(unless you got it from a fork, try `git remote -v` to check)  

`wincoe` is a small lib that I use in some of my projects on win11.  
The name comes from `wincore` but lack the `r`.  
Made with AI assistance.  

---

### Build

#### Requirements

You need `go.exe` of Go language to compile this.  
No internet required to compile, if you have Go already installed.  

#### Compile and run tests

Standard Go build:

```
go build
```

Or try `runtest.bat` to also run tests.
